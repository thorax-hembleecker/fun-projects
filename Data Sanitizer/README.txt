This is a project I embarked upon to assist in stripping my files of personally identifying information to ensure data privacy before uploading them to GitHub. The program takes a directory path as an input, then asks the user for strings representing sensitive information to search for. It then manually searches each file in the directory and returns the ones containing sensitive information. The program can either be run with detailed output, which tells the user exactly which search terms were discovered in the file and how many times they appear, or without, which runs faster but relies on the user to check the files that are returned for the exact information contained.

I'm interested in expanding this project more in the future, perhaps with a built-in GUI or the option to manually overwrite sensitive information through the program.
